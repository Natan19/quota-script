## quota-script is a simple utilitary to get the value of 1 USD in BRL
#### It can also convert from USD to BRL when a value is passed as a first parameter

#### Examples:

```shell
$ ./getquota.sh 
$ Current quota: 5.4
```


```shell
$ ./getquota.sh 1000
$ Current quota: 5.4
$ Current salary in BRL: 5400
```
